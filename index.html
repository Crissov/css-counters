Templates for CSS Counters

Additional templates for Custom Counter Styles (without a focus on i18n)

View the Project on GitHub Crissov/css-counters

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

This repository provides code templates and ideas for custom counter styles in Cascading Stylesheets (CSS) as specified in CSS Counter Styles Level 3. They can be used as list markers, for pagination, footnote markers and several other things, at least in theory. A similar, but official W3C project is Custom Counter Styles which focuses on internationalization (i18n) of enumerators, i.e. documents existing usage from around the world.

  • Classic European Counter Style Templates, some variants of roman numerals and latin alphabetic labels
  • Emoji Counter Style Templates, using colorful picture characters as list markers etc. with styles ranging from playful to serious
  • Enclosing Counter Style Templates with Combining Suffix, list markers with various shapes around them
  • Scientific (but Playful) Counter Style Templates, e.g. chemical elements, planets, days and months/zodiac signs
